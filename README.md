# hexi-photo-gallery

I want to build the best photo gallery component. I'd like it to have options for layout of photos. As far as spacing, grid type and more. I'd like it to be able to work from multiple sources like S3, local files, links, etc. I'd like this to be a solid code sample of my work that I can put on my website and be proud of

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run test` - Run tests
- `npm run lint` - Lint code
